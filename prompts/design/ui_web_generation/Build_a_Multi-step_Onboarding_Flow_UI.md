# Build a Multi-step Onboarding Flow UI

## Prompt

Design a beautiful multi-step onboarding flow: 1) Step indicator (progress bar + step labels) at top, 2) Step 1: Welcome screen with illustration and value proposition, 3) Step 2: Account creation form with social login options, 4) Step 3: Preference selection (interests as selectable chips/tags), 5) Step 4: Profile setup (avatar upload, bio), 6) Step 5: Completion screen with confetti animation and 'Get Started' CTA. Smooth slide transitions between steps. Form validation with inline error messages. Save progress to localStorage. Mobile-optimized with full-screen steps. Use a warm, inviting color scheme. Include a 'Skip' option on preference steps.
