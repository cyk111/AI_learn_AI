## Prompt

Act as a senior frontend developer and UI designer. Create a complete, production-ready landing page with the following sections: 1) Hero section with headline, subheadline, and CTA button, 2) Features grid (3 columns), 3) Testimonials carousel, 4) Pricing table (3 tiers), 5) FAQ accordion, 6) Footer with links and newsletter signup. Use modern CSS Grid and Flexbox. Include smooth scroll behavior, hover animations on buttons, and a sticky navigation bar that changes background on scroll. The design should be: clean, modern, with generous whitespace, using a professional color palette. Make it fully responsive (mobile-first). Use semantic HTML5 elements. Include subtle micro-interactions (button hover scale, card lift on hover, smooth section reveals on scroll).
