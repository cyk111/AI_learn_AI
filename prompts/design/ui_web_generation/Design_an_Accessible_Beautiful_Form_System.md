# Design an Accessible & Beautiful Form System

## Prompt

Create a comprehensive form design system with: 1) Text inputs with floating labels, icon prefixes, and character counters, 2) Select dropdowns with search/filter capability, 3) Radio button cards (visual selection with icons), 4) Toggle switches with animated transitions, 5) Checkbox cards with selection state, 6) File upload with drag-and-drop zone and preview, 7) Date picker with calendar popup, 8) Multi-step form with validation per step, 9) Inline validation (success/error states with icons and messages), 10) Submit button with loading spinner state. All inputs must be fully accessible: proper labels, aria attributes, keyboard navigation, focus indicators, error announcements for screen readers. Use a clean, modern aesthetic. Support both light and dark modes.
