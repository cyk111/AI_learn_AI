# Build a Responsive Navigation System

## Prompt

Create a modern, fully responsive navigation system with: 1) Desktop: horizontal nav with dropdown mega-menu on hover (showing columns of links), 2) Tablet: condensed nav with hamburger toggle and slide-out drawer, 3) Mobile: bottom tab bar with 4-5 main nav items + hamburger menu for secondary items, 4) Smooth transitions between states (slide, fade), 5) Current page active indicator, 6) Breadcrumb trail below nav on inner pages. The nav should have a glassmorphism effect (backdrop-filter blur) on scroll. Include a search bar that expands on click. Use CSS transitions for all state changes. No JavaScript frameworks - vanilla JS only.
