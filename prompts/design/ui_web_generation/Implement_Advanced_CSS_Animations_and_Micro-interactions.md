## Prompt

As a CSS animation specialist, create a showcase of advanced CSS animations: 1) Staggered list item entrance animations, 2) Card flip on hover (3D transform), 3) Morphing shape animation using clip-path, 4) Animated gradient border (rotating conic gradient on ::before), 5) Elastic bounce effect on buttons, 6) Skeleton loading shimmer animation, 7) Parallax tilt effect on cards (tracking mouse position), 8) Smooth page transition animations, 9) Animated SVG illustrations (stroke-dasharray/dashoffset), 10) Scroll-driven animations using CSS @scroll-timeline. Each animation should be performant (use transform and opacity only where possible, will-change, contain). Include reduced-motion media query fallbacks.
