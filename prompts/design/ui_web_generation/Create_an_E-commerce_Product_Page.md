# Create an E-commerce Product Page

## Prompt

Design a high-conversion e-commerce product page with: 1) Product image gallery with thumbnails and zoom on hover, 2) Product title, price (with sale badge if discounted), rating stars, 3) Variant selector (color swatches, size buttons), 4) Quantity selector with +/- buttons, 5) Add to cart button (prominent, animated on click), 6) Sticky add-to-cart bar on mobile, 7) Tabbed section below: Description, Specifications, Reviews, 8) Related products carousel, 9) Trust badges and shipping info. Use a clean, luxury aesthetic. Focus on conversion optimization: prominent CTAs, social proof (review count), urgency indicators (low stock). Make it pixel-perfect and fully responsive.
