## Prompt

You are a UI/UX designer specializing in B2B SaaS products. Design a comprehensive dashboard layout with: 1) Sidebar navigation with collapsible sections, icons, and active state indicators, 2) Top header bar with search, notifications bell, and user avatar dropdown, 3) KPI stat cards row (4 cards with trend indicators), 4) Main chart area with line/bar charts and date range filter, 5) Recent activity feed, 6) Quick action buttons. The design should use a professional dark sidebar + light content area scheme. Include data visualization components with proper legends and tooltips. Ensure all interactive elements have hover/focus/active states. Use a modern, clean aesthetic with consistent spacing (8px grid). Output as a complete HTML/CSS/JS implementation.
