# Create an Interactive Data Visualization Dashboard

**Category:** 🎨 Design → UI/Web 生成
**Source:** webdesignerdepot.com/25-useful-chatgpt-prompts-for-web-designers


---

## Prompt

Design an interactive analytics dashboard with: 1) Real-time metric counters with count-up animation, 2) Line chart showing trends over time with range selector, 3) Bar chart for category comparison with sort toggle, 4) Donut/pie chart for distribution, 5) Heatmap calendar (like GitHub contribution graph), 6) Data table with sorting, filtering, and pagination, 7) Export buttons (CSV, PDF). Use Chart.js or D3.js for charts. Include responsive breakpoints: 4-column on desktop → 2-column on tablet → 1-column on mobile. Add loading skeleton states. Use a dark theme with accent colors for data. Include tooltips and legend toggles on all charts. Make it feel like a premium analytics product (Stripe/Mixpanel quality).
