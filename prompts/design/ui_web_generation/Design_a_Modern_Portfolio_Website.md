## Prompt

Create a stunning portfolio website for a designer/developer with: 1) Hero section with animated typing effect and particle/3D background, 2) About section with timeline, 3) Project showcase with bento grid layout, filterable by category, and hover preview cards, 4) Skills section with animated progress bars/skill clouds, 5) Testimonials with subtle carousel, 6) Contact form with floating labels and validation, 7) Footer with social links. Use distinctive typography (no Inter/Roboto - use something like Space Grotesk or Cabinet Grotesk), creative color palette (avoid blue-purple gradients), subtle scroll-triggered animations (like AOS library), and smooth page transitions. Make it memorable and distinctive - aim for an Awwwards-level design.
